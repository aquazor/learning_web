<h1> - <h6> - элементы заголовоков
<p></p> - новая строка, абзац
<ol></ol> и <ul></ul> - упорядоченный и неупорядоченный списки
   <li></li> - элемент списка
   type="" - символьный атрибут, указывающий на тип нумерации (a,A,i,I,1)
   value="" - числовой атрибут
   reversed - обратынй порядок. Пример:
   <ol type="I">
       <li value="3">Третий элемент</li>
       <li>Четвёртый элемент</li>
       <li>Пятый элемент</li>
   </ol>
<a></a> - элемент, определяющий гиперссылку для перехода на странице или на другую старницу в интернете.
   href - обязательный атрибут для определения ссылки
   <a href="ссылка">ТЕКСТ</a>
   id - атрибут id
   # - символ якоря, который указывает на элемент id на страницу
   <p id="par1">
   <a href="#par1">Перейти к пар1</a>
